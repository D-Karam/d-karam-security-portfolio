# CTF Writeups

This directory will contain various write-ups for Capture The Flag (CTF) challenges. Each write-up will follow a template that ensures consistency and clarity in documentation.

## Template Format
1. **Challenge Name**: The name of the challenge solved.
2. **Category**: The category of the challenge (Web, Pwn, Crypto, etc.).
3. **Description**: A brief overview of the challenge.
4. **Solution**: Detailed steps taken to solve the challenge.
5. **Tools Used**: Any tools that were utilized during the process.
6. **Conclusion**: A summary of what was learned from the challenge.

Feel free to contribute by adding your own write-ups and following the above template!
